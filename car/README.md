# Car Code

Car-side embedded code lives here.

Create a separate English-named folder for each module or experiment. Current modules:

- `bluetooth_uart_test/` - Bluetooth serial bring-up and echo test code.
- `line_tracking_manual/` - manual-push grayscale line tracking test code.
- `oled_ssd1306/` - MSPM0 OLED I2C display test code.
